Open-source dictation for your Mac

Hold a key, talk, and the transcript is pasted into whatever app has focus. One HTTPS call to AssemblyAI's speech-to-text API — typically back in under 100 ms — with no local models, no subscription, and no second LLM pass. Bring your own API key.

Features

The details

Zero dependencies

Plain Swift on Apple's frameworks. No external packages — if it isn't in the repo or the SDK, it isn't in the binary.

No telemetry

No crash reporting, no analytics, no update pings. The only network call is the transcription request, and the code that makes it is right there in the repo.

Native Mac app

AppKit and SwiftUI, a CGEventTap for the hotkey, a paste into the focused app. No Electron, no web views, no background agents.

Accurate transcription

Audio goes to AssemblyAI's Sync STT API with a prompt built from the app you're typing into, and comes back already cleaned up — one round trip, no post-processing.

18 languages

Dictate in 18 languages, not just English. Pick one in settings and go.

Real synth cues

The start and stop sounds are sampled from an actual Yamaha DX7 and Roland Juno-106. There's an off switch, but why would you.
